## 1. Key Principles of Use

### What it is

Button Group — группа связанных кнопок, объединённых в единый контрол. Используется для переключения режимов или выбора одного из взаимоисключающих действий. В отличие от отдельных Button, кнопки в группе визуально соединены и воспринимаются как единый элемент.

### When to use

**Use** — когда нужно предоставить несколько взаимосвязанных действий или режимов: переключение вида (список / сетка), выбор выравнивания текста, набор фильтров с единственным активным выбором.

**Don't use:**
- Для независимых, несвязанных действий — используйте отдельные **Button**
- Для переключения между view с контентом — используйте **Tabs**
- Для фильтрации с множественным выбором — используйте **Segmented Control** или чекбоксы
- Более 5 кнопок в группе — разбейте на отдельные контролы или используйте Dropdown

### Core principles

- **Смысловая связь** — кнопки в группе должны быть семантически связаны и относиться к одной задаче
- **Один активный** — в группе переключения всегда активна ровно одна кнопка
- **Единый размер** — все кнопки в группе одного размера

│ Button 1 │ Button 2 │ Button 3 │
└──────────┴──────────┴──────────┘

├──────────┤

| Slot | Обязательность | Описание |
|---|---|---|
| `button-item` | required (min 2) | Дочерняя кнопка. Наследует пропы от Button |
| `divider` | auto | Разделитель между кнопками. Генерируется автоматически |

## 3. Types / Variants

| Тип | Назначение |
|---|---|
| `horizontal` | Кнопки расположены в строку. Дефолт |
| `vertical` | Кнопки расположены в столбец. Для боковых панелей |

### Modifiers

| Модификатор | Описание | Ограничения |
|---|---|---|
| `equal-width` | Все кнопки одинаковой ширины | — |
| `stretch` | Группа занимает всю ширину контейнера | Только `horizontal` |

## 4. Sizes

Наследует глобальную систему размеров SEDA UI через prop `size`, применяемый ко всей группе.

| Size | Height | Font / Line | Radius (крайние) | Контекст |
|---|---|---|---|---|
| `small` | 24px | 12px / 16px | 6px | Тулбары, компактные панели |
| `medium` | 32px | 14px / 20px | 8px | Формы, карточки — дефолт |
| `large` | 40px | 16px / 24px | 10px | Акцентные переключатели |
| `extraLarge` | 48px | 18px / 28px | 12px | Hero-секции |

> Внутренние кнопки не имеют скругления на смежных сторонах — только крайние кнопки группы скруглены снаружи.

## 5. States

### State matrix

Состояния наследуются от компонента Button и применяются к каждой кнопке индивидуально.

| Состояние | Описание | Применение |
|---|---|---|
| `default` | Исходный вид | Ко всей группе |
| `hover` | Курсор над кнопкой | К отдельной кнопке |
| `active` | Нажатие (mousedown) | К отдельной кнопке |
| `focus` | Фокус клавиатуры | К отдельной кнопке |
| `selected` | Активный режим в группе переключения | К отдельной кнопке |
| `disabled` | Недоступна | К отдельной кнопке или всей группе |

### Valid state combinations

| Комбинация | Допустимо | Примечание |
|---|---|---|
| `selected` + `hover` | ✓ | Наведение на активную кнопку |
| `selected` + `focus` | ✓ | Фокус на активной кнопке |
| `disabled` (одна кнопка) | ✓ | Остальные остаются интерактивными |
| `disabled` (вся группа) | ✓ | Применяется через prop `disabled` на группе |
| `selected` + `disabled` | ✗ | Выбранная кнопка не может быть недоступна |

## 6. Details on Types / Variants

### horizontal
Кнопки выровнены по горизонтали. Смежные границы объединяются: между соседними кнопками одна общая граница шириной 1px. Первая кнопка скруглена слева, последняя — справа.

### vertical
Кнопки выровнены по вертикали. Ширина группы определяется самой широкой кнопкой или контейнером. Первая кнопка скруглена сверху, последняя — снизу.

## 7. Behavior

### Keyboard interaction

| Клавиша | Действие |
|---|---|
| `Tab` / `Shift+Tab` | Перемещение фокуса между кнопками |
| `Enter` / `Space` | Активация кнопки в фокусе |
| `→` / `↓` | Переход к следующей кнопке (если `role="radiogroup"`) |
| `←` / `↑` | Переход к предыдущей кнопке (если `role="radiogroup"`) |

### Режим переключения vs режим действий
- **Переключение (toggle)** — группа ведёт себя как `radiogroup`. Стрелки навигируют между кнопками, выбранная остаётся активной.
- **Действия** — каждая кнопка независима. Tab-навигация, Enter/Space активируют.

## 8. Accessibility

### Keyboard
При использовании как переключатель: `role="radiogroup"` на группе, `role="radio"` на каждой кнопке. При использовании как набор действий: стандартная Tab-навигация.

### Screen reader

| Атрибут | Значение | Когда |
|---|---|---|
| `role="group"` | — | Набор независимых действий |
| `role="radiogroup"` | — | Группа переключения (один активный) |
| `aria-label` | Описание группы | Когда нет видимого заголовка |
| `aria-checked="true"` | — | Активная кнопка в `radiogroup` |
| `aria-pressed="true"` | — | Активная кнопка в toggle-режиме |

### Visual
- Контрастность: минимум 4.5:1
- Focus ring на каждой кнопке индивидуально
- Активная кнопка не передаётся только цветом
