# Модель Валидации

## Когда Валидировать
- После взаимодействия пользователя с полем
- При отправке формы

## Состояния
- `error` → блокирует действие
- `warning` → предупреждает, но не всегда блокирует
- `success` → подтверждает корректность

## Правила
- Не показывайте ошибку на пустом поле, с которым пользователь ещё не взаимодействовал.
- Всегда добавляйте текст сообщения, а не только цвет или иконку.
