Button

Actions v1.0 SEDA UI

What it is

Button — основной элемент взаимодействия. Инициирует действие: отправку формы, подтверждение операции, переход к следующему шагу. В отличие от Link, Button не ведёт к ресурсу — он запускает процесс.

When to use

Use
Когда нужно инициировать действие: отправить форму, подтвердить удаление, запустить процесс, сохранить изменения.
Don't use
Когда нужна навигация к URL — используйте Link. Когда нужен выбор из вариантов — используйте Radio или Segmented Control.

Core principles

Один Primary на контекст. Primary Button задаёт главное действие экрана. Несколько Primary одновременно размывают фокус.

Иерархия через тип. Primary → Secondary → Ghost → Text отражает убывающую визуальную важность. Комбинируйте типы, чтобы расставить приоритеты.

Destruction только для необратимого. Красный цвет — сигнал опасности. Используйте только там, где действие нельзя отменить.

Loading behavior

При переходе в loading: кнопка блокируется, иконка заменяется спиннером, label остаётся видимым. Ширина кнопки не меняется — спиннер занимает место иконки или добавляется слева от label.

Click zone

Минимальная интерактивная зона — 44×44px (WCAG 2.5.8). Для small и medium зона клика расширяется прозрачным padding: визуальный размер кнопки не меняется.

Full-width

Модификатор full-width применяется только к одиночной кнопке в контейнере. Для группы кнопок используйте Button Group.

Accessibility

Keyboard
Фокусируется через Tab. Активируется через Enter и Space. В состоянии disabled исключается из tab-order через нативный атрибут disabled, не только визуально.
Screen reader
Роль role="button" несёт нативный <button>. Для icon-only: обязателен aria-label. В состоянии loading: добавить aria-busy="true". Состояние disabled: использовать атрибут disabled, а не aria-disabled.
Visual
Контрастность текста на фоне кнопки: минимум 4.5:1 (WCAG AA). Focus indicator не зависит только от цвета — кольцо focus/ring видимо на любом фоне. Состояние disabled: контраст ниже 4.5:1 допустим (WCAG 1.4.3).
Touch
Минимальный touch-target: 44×44px (WCAG 2.5.8). Для small и medium — расширяется невидимым padding или псевдоэлементом.
